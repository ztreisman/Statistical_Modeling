Sure, no problem Jonathan.

Two things I'll add for you and/or Zack - I didn't look at correlations among the different spectral indices, but I would expect that a lot of the 2013-2018 Landsat time series predictors are pretty strongly related to one another (the exception probably being the winter NDVI variable which is a bit different since it is capturing changes in greenness in a different season than the rest). And probably the same thing is true for Jianmin's phenology metrics - I would expect that things like start of season, end of season, and season length are pretty strongly related. For a clearer interpretation in the paper, it could be good to reduce the LTS predictors and phenology predictors down a bit. Figured I'd just include them all now for simplicity.

Also, if we want to use the fitted statistical models to predict across the landscape, the easiest way to do that is by fitting the model in R and saving the fitted model as an .rds object. Ideally, the model/package used to fit the model will have a "predict" function associated with it as well - the raster package works a lot more smoothly for making spatial predictions in those cases.

Sorry for a bit of radio silence the last couple of weeks! Turned attention to a few other things, but had a chance to get back this analysis a bit today. All changes have been pushed to GitHub including updated data and new spatial datasets.

Btw, I think I now have a plan for extracting some Landsat time series stuff to look at post-disturbance trajectories as well - using LandTrendr for the 2014-2018 period with a single segment, which basically fits a linear regression to the vegetation trend in that period to get the slope of change, rather than the time series plots I showed you. Those longer time series plots are essentially a piecewise linear regression with multiple line segments, but a single segment is enough to get post-disturbance trend. Starting at 2014 makes sure that all of the pixel values start post-fire, because the 2013 pixel value could be pre- or post-fire depending on clouds in different scenes.
